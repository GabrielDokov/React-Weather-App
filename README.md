# React-Weather-App

This is a weather app, which shows daily and weekly weather for current city

# Technologies

 - ReactJS
 - HTML
 - CSS
 - API - (one API to get cities, and one to get Weather)
 
 
 # Functionality
 
  - Daily weather forecast
  - 7-day weather forecast
  - Search city by name
